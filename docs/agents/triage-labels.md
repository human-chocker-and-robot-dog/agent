# Triage labels

Engineering skills use five canonical triage roles. This table maps them to the labels used by this repository.

| Canonical role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer evaluation is required. |
| `needs-info` | `needs-info` | Waiting for more information from the reporter. |
| `ready-for-agent` | `ready-for-agent` | Fully specified and ready for an autonomous agent. |
| `ready-for-human` | `ready-for-human` | Requires human implementation. |
| `wontfix` | `wontfix` | Will not be actioned. |

When a skill refers to a canonical role, apply the corresponding GitHub label. `wontfix` already exists; the remaining labels are this repository's agreed vocabulary and should be created in GitHub before they are applied.
